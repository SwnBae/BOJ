리스트에서 맨 앞의 요소 가져오기?

1. ArrayList
--> 맨 앞의 요소를 빼고, 뒤에 있던 모든 요소를 복사해서 가져와야 하기 때문에 비효율 O(n)
--> index를 바탕으로 검색하기 때문에, 빠른 검색 가능 O(1)

2. LinkedList: 포인터 참조로 연결되어있음. 각 노드는 데이터 저장
 **이중 연결 리스트(Double-Linked List)**로 구현되어 있으며, 각 노드는 다음과 같은 두 가지 주요 정보를 가지고 있습니다:
  1)데이터(value)
  2)앞/뒤 노드를 가리키는 포인터 (prev, next)

이후, 연결이 끊긴 노드는 Garbage Collector가 정리해준다.

--> 맨앞, 맨뒤 요소 삽입/삭제는 O(1), [단, 중간 삽입/삭제는 O(n)]
--> 값을 검색할 때는 처음부터 순차적으로 포인터를 따라가며 탐색 O(n)

--> poll(맨앞), push(맨뒤)



★ 삽입/삭제가 중요할 경우 LinkedList, 검색이 중요할 경우 ArrayList

ex) Queue: LinkedList
